Questions:

What is JavaScript? How did it get its name?

JavaScript is a programming language used to make websites interactive. It allows things like forms, animations, and real-time updates without needing to reload the page.

What are the two ways you can run JavaScript?

In the browser: JavaScript runs directly in our web browser when we visit a website. On the server: JavaScript can also run on a server using a tool called Node.js.

What was the most popular front-end js library?

Before modern tools, jQuery was the most popular JavaScript library. It made things like animations and clicking easier.

What is the most popular front-end js library today?

Today, React is the most popular JavaScript library for building websites, followed by Vue.js. React helps developers create interactive user interfaces easily.
