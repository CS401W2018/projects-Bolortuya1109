Box Model

What is the difference between block tags and inline tags?

Take up the full width and start on a new line (e.g., div, p, h1). Only take up as much space as needed and stay in line (span, a, strong).

What is the margin, border, and padding?

Margin: Space outside the border. Border: The edge of the element. Padding: Space inside the border, around the content.

What are units of measurement you can use in CSS?

Absolute: px, cm, mm, in (fixed size). Relative: %, em, rem, vw, vh (size changes based on other elements or screen size)

When would you use a figure tag?

Use it for images, charts, or code snippets with a caption
