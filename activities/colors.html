What is the difference between an ID and a Class?

A class name can be used by multiple HTML elements, while an id name must only be used by on HTML element within the page.

What is the difference between a SPAN tag and a DIV tag?

The DIV tag is a generic block-level element used for meaning it starts on a new line and stretches across the full width available. SPAN is an inline element, meaning it does not start on a new line and only takes up as much width as necessary.

What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

DIV is a generic block-level container with no inherent meaning. Header, main, and footer are semantic elements that clearly define the structure of a webpage, improving readability and accessibility.

What are the different ways to select a color?

Named color uses predefined color names. Hexadecimal code is a six-character code representing RGB values. RGB Defines colors using red, green, and blue values.
