What is AJAX?

AJAX stands for Asynchronous JavaScript and XML. It allows web pages to load and update data without refreshing the entire page. This makes websites feel faster and more interactive.

Why do you put your JS Code in a separate file?

We put our JavaScript in a separate file because it keeps our code organized and clean. It makes our HTML file less cluttered. It allows reusability of our code across different pages. It can improve performance because the browser can cache the JavaScript file.
