Internet

The internet is an international computer network connecting other networks and computers that allows people to share information around the world.

Web

A service that operates on the internet which allows access to websites and online content through the browsers.

How are the Internet and Web different?

The internet is the network, and the web is a service on the internet.

ASCII and Unicode

ASCII is a basic text encoding system. Unicode is to support global characters and symbols.

What is UTF-8

UTF-8 is a flexible encoding format for Unicode that is compatible with ASCII.

URL

The address used to access resourses on the web, like a website.

Domain Name

A readable address that maps to IP address.

HTTP

A protocol for transferring data on the web.

HTTPS

HTTPS is the secure version of HTTP with encrypted data.

IP Address

A unique number identifying devices on the network.

DNS

A Domain Name System (DNS) turns domain names into IP addresses, which allow browsers to get to websites and other internet resources.

DW3C Recommendations

Standards for web technologies by the world wide web consortium.

XML, XHTML, HTML, HTML5

These are languages for structuring and displaying web content with HTML5 being the latest.
